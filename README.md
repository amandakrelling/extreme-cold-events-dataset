# extreme-cold-events-dataset

Scripts for detecting cold snap events and calculating associated metrics from historical weather data across multiple U.S. cities.

## Repository Structure

### main_code.py
Runs the main analysis workflow, including weather data preprocessing, cold snap event detection, and event characterization.

### preprocessing.py
Preprocesses historical weather data to generate formatted inputs for cold snap event detection.

### characterize_coldsnap_code.py
Computes metrics and characteristics for detected cold snap events.

### event_selection_code.py
Selects a subset of detected cold snap events to construct the final dataset.
